# Minesweeper 2P 🎮

Minesweeper 2 ผู้เล่น ออนไลน์ — เล่นผ่าน WebRTC (PeerJS) ไม่ต้องมีเซิร์ฟเวอร์!

## วิธีเล่น

**https://xupiter01.github.io/minesweeper/**

### สร้างห้อง
1. กด **🎮 สร้างห้องใหม่**
2. แชร์ลิงก์หรือรหัส 4 ตัวให้เพื่อน
3. เลือกที่นั่ง **🟦 ซ้าย** หรือ **🟥 ขวา**
4. กดยืนยัน — เกมเริ่ม!

### เข้าร่วม
1. เปิดลิงก์ที่เพื่อนแชร์ หรือกรอกรหัส 4 ตัว
2. รอเจ้าห้องเลือกที่นั่ง
3. เกมเริ่มอัตโนมัติ

## กติกา

- คลิกซ้าย ➔ เปิดช่อง
- คลิกขวา ➔ ปักธง
- **ผู้เล่นแต่ละคนมีคะแนนของตัวเอง** — ทุกครั้งที่เปิดช่องได้ จะได้คะแนนตามจำนวนช่องที่เปิด × combo multiplier
- เหยียบระเบิด ➔ **แพ้ทันที** (คู่ต่อสู้ชนะ)
- เปิดช่องครบ ➔ **คนที่คะแนนมากกว่าชนะ**
- ระบบซิงค์แบบ Real-time ผ่าน WebRTC

## ⚙️ ความยาก

เลือกได้ 3 ระดับ:
- 🟢 **ง่าย** — 6×6, 5 ระเบิด
- 🟡 **กลาง** — 9×9, 10 ระเบิด (default)
- 🔴 **ยาก** — 14×14, 30 ระเบิด

## 💣 Power-ups (ผู้เล่นละ 1 ครั้ง/เกม)

- 🎯 **สแกน** — เปิดช่องสุ่ม 1 ช่องที่ปลอดภัย
- 🛡️ **โล่** — ป้องกันการเหยียบระเบิด 1 ครั้ง (โล่หายหลังใช้)
- 🔍 **เปเปอร์** — คลิกช่องใดก็ได้ → เปิด 3×3 รอบข้าง

## 🔥 Combo System

คลิกเปิดช่องติดกันภายใน 1.5 วินาที = combo เพิ่มคะแนนคูณ:

| Combo | Multiplier | Label |
|------:|-----------:|-------|
| 3 ครั้ง | ×1.5 | ✨ COMBO! |
| 5 ครั้ง | ×2 | ⚡ UNSTOPPABLE! |
| 8 ครั้ง | ×3 | 🔥 ON FIRE! |

หยุดเกิน 1.5 วิ → reset combo

## เทคโนโลยี

- **PeerJS** — WebRTC data channel สำหรับเชื่อมต่อ P2P
- **GitHub Pages** — โฮสต์ไฟล์ static
- ไม่ต้องมีเซิร์ฟเวอร์แบ็กเอนด์!
